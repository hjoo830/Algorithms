## 문제

두 개의 정수를 입력 받아 두수를 더한 결과를 출력하시오.​

## 입력

두 정수 a, b를 입력 받는다. (1 ≤ a, b ≤ 1,000)

## 출력

a+b 결과를 출력하시오.

## 예제

### 입력1

```
1 2
```

### 출력1

```
3
```

### 입력2

```
3 4
```

### 출력2

```
7
```
